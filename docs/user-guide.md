# 事務ツール 使い方ガイド

請求書・領収書・見積書・支払明細書・納品書を PDF で発行できるデスクトップアプリです。
ローカル完結型なので、データはすべてあなたの PC 内に保存されます。

---

## 1. このツールでできること

- 5 種類の書類を作成・PDF 出力
  - 請求書、領収書、見積書、支払明細書、納品書
- マスター登録で入力を効率化
  - 会社情報（自社）
  - 取引先情報（お客様・発注先）
  - 品目情報（商品・サービス）
  - 印影画像（角印）
- 書類番号の自動採番（取引先ごとの個別採番にも対応）
- 消費税計算（10% / 軽減税率 8%）、源泉徴収の自動計算
- 書類履歴から過去の書類を複製・再利用
- データ保存先をクラウドフォルダ（OneDrive 等）に設定して PC 間で同期

## 2. インストール

配布ページ: https://github.com/Aizoon-jp/document-tool/releases/tag/v0.1.5

### Windows の場合

1. `Setup.0.1.5.exe` をダウンロード
2. ダブルクリックしてインストーラーを起動
3. 指示に従ってインストール（30 秒程度）
4. スタートメニューに「事務ツール」が追加されます

### Mac の場合

1. お使いの Mac に合わせてファイルをダウンロード
   - Apple Silicon (M1 / M2 / M3 / M4): `事務ツール-0.1.5-arm64.dmg`
   - Intel Mac: `事務ツール-0.1.5.dmg`
2. ダウンロードした `.dmg` ファイルをダブルクリック
3. 表示されたウィンドウで、事務ツールを「アプリケーション」フォルダにドラッグ
4. アプリケーションフォルダから事務ツールを **右クリック → 開く**
   （通常のダブルクリックだと「開発元が確認できません」と表示されてしまうため）
5. 確認ダイアログで「開く」を選択
6. 次回以降は普通にダブルクリックで起動できます

### Linux の場合

1. `事務ツール-0.1.5.AppImage` をダウンロード
2. ファイルに実行権限を付与：`chmod +x 事務ツール-0.1.5.AppImage`
3. ダブルクリックまたはターミナルから実行

## 3. 初回起動と全体の流れ

アプリを起動すると、左側にメニュー、右側に作業画面が表示されます。

左側メニューの項目：

- **ダッシュボード** — 月別の書類件数などのサマリー
- **書類作成** — 各種書類の新規作成
- **書類履歴** — 過去に作成した書類の一覧・検索
- **設定** — 各種マスター情報の登録・変更

### おすすめの初期セットアップ順

1. 設定 → 会社情報
2. 設定 → 取引先
3. 設定 → 品目
4. 設定 → 印影
5. 書類作成へ進む

一度登録してしまえば、以降の書類作成はポチポチ選ぶだけで完成します。

| 住所 | | |
| 電話番号 | | |

## 5. 取引先を登録する

書類の宛先となる取引先を登録します。複数登録できます。

**登録手順**
1. 設定 → 取引先タブ
2. 右上の「新規追加」をクリック
3. 以下の項目を入力して「保存」

**入力項目**

| 項目 | 必須 | 備考 |
|------|:---:|------|
| 取引先名 | ○ | 正式名称 |
| 敬称 | ○ | 御中 or 様 |
| 郵便番号 / 住所 | | |
| 電話番号 | | |
| 担当者 / 担当部署 | | |
| 支払条件 | | 例: 月末締め翌月末払い |
| デフォルト税区分 | | 課税 10% / 課税 8%（軽減）/ 非課税 |
| **書類番号プレフィックス** | | 重要項目、下記参照 |
| 備考 | | |

### 書類番号プレフィックスについて

取引先ごとに書類番号の体系を分けたい場合に使います。
空欄のままでも問題ありません。

**パターン 1: 何も入力しない（標準）**
- 書類番号: `INV-202604-0001`、`INV-202604-0002`、...
- 会社全体で通し番号

**パターン 2: プレフィックスを入力する（例: `A-`）**
- 書類番号: `A-INV-202604-0001`、`A-INV-202604-0002`、...
- 取引先ごとに独立した採番になる
- 他の取引先に `B-` を設定すると、B 取引先は `B-INV-...` で別採番

**パターン 3: 最初の 1 枚を手入力する**
- プレフィックス欄は空のまま登録
- 書類作成時に書類番号欄を手動で書き換える（例: `CUSTOM-2026-001`）
- 次回以降、同じ取引先で書類を作る時は自動で末尾の数字が +1 された番号が提案される

## 6. 品目を登録する

よく使う商品・サービスを品目として登録しておくと、書類作成時に選ぶだけで入力完了します。

**登録手順**
1. 設定 → 品目タブ
2. 「新規追加」をクリック
3. 以下の項目を入力して「保存」

**入力項目**

| 項目 | 必須 | 備考 |
|------|:---:|------|
| 品目名 | ○ | 例: 月額顧問料 |
| 単価 | ○ | 数値（円） |
| 単位 | ○ | 式 / 個 / 時間 / 月 など自由記載 |
| 税率 | ○ | 10 / 8 / 0（非課税） |
| 軽減税率適用 | | 8% の場合にチェック |
| デフォルト数量 | | 書類作成時の初期値 |
| 備考 | | |

### 品目登録のコツ

- 金額が毎回変わる商品でも「単価 0 円」で登録しておけば、書類作成時に上書き入力できます
- 品目名が変動するものも同様。マスターになくても、書類作成時に品目名を自由記載できます

---

## 7. 印影を登録する

角印の画像を登録すると、書類の PDF に自動で合成されます。

**事前準備**
- 角印の画像ファイル（PNG または JPG、5MB 以下）
  - **背景が透明な PNG** を推奨（不自然な白い四角が出ない）
  - サイズは 300〜600px 四方程度が目安
- 角印が手元にない場合は、文具店で作るか、オンラインの角印作成サービスで作成してください

**登録手順**
1. 設定 → 印影タブ
2. 「新規追加」をクリック
3. 印影名（例: 会社印）を入力
4. 画像ファイルを選択
5. 表示位置・サイズ・透明度を調整
6. 「既定の印影にする」をチェックすると、書類作成時に自動で選択されます
7. 「保存」

**表示位置のコツ**
- 書類右上の代表者名の近くに配置するのが一般的
- 透明度 80% が自然な押印風に見えます
- 複数の印影を登録して、書類ごとに切り替え可能

## 8. 書類を作成する

いよいよ書類作成です。取引先・品目マスターが揃っていれば、数クリックで完成します。

**手順**
1. 左メニューの「書類作成」をクリック
2. 書類種別を選ぶ（請求書 / 領収書 / 見積書 / 支払明細書 / 納品書）
3. 以下を入力

**入力項目**

| 項目 | 備考 |
|------|------|
| 書類種別 | 最初に選んだもの、あとから切り替え可 |
| 取引先 | マスターから選択（選ぶと書類番号が自動採番） |
| 発行日 | デフォルトは本日 |
| 書類番号 | 自動採番。手動編集も可能 |
| 明細（直接記載 / 別紙明細の通り）| どちらかを選択 |

### 明細の入力（直接記載モード）

1. 「明細行を追加」ボタンで行を追加
2. 各行に以下を入力
   - 品目名（マスターから選ぶ or 自由記載）
   - 数量、単位、単価
   - 金額は自動計算されます
3. 不要な行は右上の「削除」ボタンで消せます

### オプションの設定

以下のオプションをトグルで切り替えられます。

- 消費税を表示
- 軽減税率（8%）を適用
- 源泉徴収を自動計算（業務委託の報酬等で利用）
- 備考を表示
- 振込先情報を表示（請求書・支払明細書はデフォルトでON）

### 印影の選択

印影マスターに登録した印影から、使用するものを選択します。
既定の印影が自動で選ばれますが、書類ごとに変更できます。

---

## 9. PDF を出力する

1. 書類作成画面で内容を入力
2. 画面右側のプレビューで見た目を確認
3. 画面右上の「PDF 生成」ボタンをクリック
4. 保存先を選択すると PDF が生成される

### PDF の保存先について

初期設定では以下に保存されます。

- Windows: `C:\Users\<ユーザー名>\Documents\事務ツール\`
- Mac: `/Users/<ユーザー名>/Documents/事務ツール/`

保存先を変更したい場合は、設定 → データ保存先 から変更できます。

### 下書き保存

PDF を生成せず、書類データだけ保存したい場合は「下書き保存」ボタンを使います。
後で書類履歴から開いて編集・PDF 生成できます。

---

## 10. 書類履歴と複製

過去に作成した書類は「書類履歴」で一覧できます。

- 取引先名 / 書類種別 / 発行日 / 金額で絞り込み検索
- 各書類から「複製」を実行すると、同じ内容で新しい書類が自動作成されます
  - 日付と書類番号は自動で新しいものに
  - 月次で定期請求する取引先などに便利

---

## 11. データの保存先とバックアップ

### データ保存先の確認・変更

1. 設定 → データ保存先
2. 現在の保存先が表示されます
3. 変更したい場合は「フォルダを選択」で新しい場所を指定

### PC 間で同期したい場合

OneDrive や iCloud Drive のようなクラウド同期フォルダを保存先に指定すれば、
別の PC からでも同じデータで作業できます。

**設定例（OneDrive の場合）**
- 保存先を `C:\Users\<ユーザー名>\OneDrive\事務ツール` に設定
- 別の PC でも同じツールをインストールし、同じ OneDrive フォルダを保存先に指定
- 双方のデバイスで同じデータが見える

### バックアップ

データは以下の場所にあります。
ファイル一式を定期的に外付け HDD や別のクラウドにコピーするだけでバックアップ完了。

- データベース本体: `data.db`
- 印影画像: `stamps/` フォルダ

## 12. よくある質問とトラブル

### Q. Mac で「開発元が確認できません」と表示されて開けない

.dmg 内のアプリケーションアイコンを **右クリック → 開く** で回避できます。
1 回目だけこの操作が必要で、2 回目以降はダブルクリックで普通に起動します。

### Q. PDF 出力したら日本語が文字化けする

通常は発生しない想定ですが、もし起きた場合は一度アプリを再起動してください。
それでも直らない場合は GitHub の Issues で報告してください。

### Q. データを消してしまった

万が一に備えて、`data.db` ファイルのバックアップを定期的に取ることを推奨します。
消してしまった場合、バックアップがあればそのファイルを元の場所に戻すだけで復元できます。

### Q. 書類番号を途中で変更したい

書類作成画面の「書類番号」欄は手動編集可能です。
変更した番号は次回以降の自動採番にも影響します（取引先ごとに）。

### Q. 新しい書類種別を追加したい

標準の 5 種類以外の書類種別を追加したい場合は、開発者または
Claude Code 等の AI アシスタントに相談してください。

### Q. 会社ロゴを書類に入れたい

現バージョンでは印影としてロゴ画像を登録すれば同様に表示できます。
本格的なロゴ配置機能は将来のバージョンで対応予定です。
